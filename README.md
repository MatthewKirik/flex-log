# Flex Log

**It's a simple logging library**

## Quick Start

```
$ git clone https://github.com/MatthewKirik/flex-log.git
```

```
$ cd flex-log
```

```
$ npm i
```

## Project contents

### The project provides and uses:

- Telegram API
- Console logging into STDOUT stream
- File logging
